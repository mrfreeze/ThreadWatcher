0.3.5 Changes
  • Added support for fetching thread URLs from Opera and Camino
  • Newer versions of Growl, Appscript and BWToolkit frameworks
  • The positions of the dividers in the watcher window now get saved
  • Fixed several problems when using script commands to save images
  • The saveSheet script parameter now opens the sheet immediately
  • Created some new example scripts (see http://mrfreeze.github.com/ThreadWatcher/scripts.html)

0.3.4 Changes

Small update to fix a few bugs.

  • Fixed bug that allowed tabs to be closed while a sheet was open (which prevented windows from closing)
  • Fixed posts being cut off in full thread view.
  • Minor changes to toolbar
  • Fixed throbber not stopping after an error fetching a thread
